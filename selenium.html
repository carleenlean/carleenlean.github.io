SeleniumLibrary 包含瀏覽器基礎使用的關鍵字。行為包含：尋找頁面指定元素，開啟瀏覽器，截圖，執行 Javascript，滑鼠互動，回傳字串等。。。

**關鍵字過多時，若有指定要搜尋的關鍵字，建議按 Ctrl + F 直接搜尋相關文字。

  • 使用引述：關鍵字/警示視窗文字/設定等待警示視窗出現的時間
    Alert Should Be Present 您已經登入成功 5s
  • 使用引述：關鍵字/警示視窗文字/取消警示視窗/設定等待警示視窗出現的時間
    Alert Should Be Present 您已經登入成功 DISMISS 5s
Handle Alert (處理警告視窗) action=ACCEPT (警示視窗處理), timeout=None（設定等待警示視窗出現的時間）

視窗出現後有三種處理方法：確認視窗 (ACCEPT)，取消視窗 (DISMISS) 或是讓視窗維持開啟狀態 (LEAVE)。

  • Handle Alert (接受警告訊息)
  • Handle Alert DISMISS 10s (等待 10秒讓警告訊息出現後取消警告訊息)
  • ${message}= Handle Alert LEAVE (把警告視窗的訊息存到 message 裡面並 使用 LEAVE 讓警告視窗維持開啟狀態)
Alert Should Not Be Present (不應該出現警告視窗) action=ACCEPT (接受警示視窗), timeout=0（設定等待警示視窗出現的時間） Alert Should Not Be Present 5s
Capture Element Screenshot (擷取元素圖示) locator (元素定位), filename=selenium-element-screenshot-{index}.png (截圖的檔案輸出位置與檔名）
  • 必要引述：locator (元素定位)
  • 設定圖檔名稱自動增加索引
    Capture Element Screenshot xpath=//*[@id="UHSearch"]/table/tbody/tr[2]/td/ul ../tutorial/element_show_{index}.png
    “../tutorial/” 指定檔案存入位置 "element_show_{index}.png" 指定圖示檔案名稱。{index} 可以幫指定名稱的圖檔自動增加一個索引，這樣可以預防既有的圖檔被覆蓋。
  • 設定圖檔指定名稱
    Capture Element Screenshot xpath=//*[@id="UHSearch"]/table/tbody/tr[2] test/imagetest_capture.png
  • 如果沒有設置截圖輸出入徑，輸出的檔案就會直接在目前所在的資料夾建立。
Capture Page Screenshot (擷取頁面) filename=selenium-screenshot-{index}.png（指定圖示輸出檔案名稱並且加索引）
  • Capture Page Screenshot img/testbutton-{index}.png (指定檔案存的入徑 (img/)，並指定圖檔名稱 (testbutton-) 新增索引 ({index})
  • Capture Page Screenshot (未指定入徑時檔案直接存入執行檔所在位置)

    Execute Javascript (執行 Javascript) *code

    要使用這個關鍵字需要對 Javascript DOM 有一定的瞭解。下面列出幾個範例:

    • 找尋頁面指定元素 id "ColorButton" 狀態改為非啟用
      Execute Javascript document.getElementById("ColorButton").disabled=false
    • 選指定元素 id "ProductName" 裡面的 "jquery"
      Execute Javascript $("#ProductName").val("jquery")
    • 按指定元素 id "ProductName"
      Execute Javascript $("#ProductName").click()
    • 開啟一個指定網址的新視窗
      Execute Javascript window.open()
    Input Password (輸入密碼) locator（元素定位）: 指定元素位置

    text (文字)： 輸入的文字
    clear=True (清除)： 輸入文字之前清除欄位既有文字     		Input Text xpath=/div/div/text user01

    如果沒有指定瀏覽器，執行時就會使用預設的瀏覽器執行。
