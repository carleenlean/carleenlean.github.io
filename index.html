Robot Framework 介紹

RF 介紹 RF 安裝 RF 概念

Robot Framework 是一個不需要有任何程式語言基礎的自動化工具。讓沒有程式語言背景的測試工程師跨越艱澀的程式語言邏輯跟訓練也可以順利的讓測試達到自動跑起來的目的。

建立 Robot Framework 自動化教學的初衷，是由於許多人想要從手動測試工程師轉為自動化測試工程師。卻因為找不到適合自己的資料而無法順利學習。本站希望可以藉由這個網站降低學習的門檻並且幫助想要擁有多項技能的測試工程師成長進而學習。

讓我們開始學起來吧！！！ 來～來～ 接下來讓我講講 Robot Framework 的運用邏輯是什麼吧～ 你可以把它想像成... 一個指令一個動作的操作邏輯。假設我們要用瀏覽器打開一個網頁。這個時候就會告訴電腦，使用關鍵字 Open Browser 打開網址 https://www.yahoo.com.tw 然後使用 chrome 瀏覽器開啟。最後再使用關鍵字Close Browser 關掉瀏覽器。

完整的腳本就像下面的範例呈現：

  • Open Browser    https://www.yahoo.com.tw    chrome      ⇥開啟 chrome 瀏覽器並打開指定網站⇤
  • Close Browser      ⇥關閉瀏覽器⇤

看起來不會很難吧？！ 它難就是難在如何正確的理解以及正確使用各個不同的關鍵字。講到這裡，是不是不會太難呢？！也許它也可以變成你選擇自動化的其中一個工具喔！！！

以下是比較常用到的官方標準資料庫：

資料庫 資料庫敘述
SeleniumLibrary 打開/關閉瀏覽器 輸入/頁面文字存在確認
BuiltIn 迴圈使用 重複關鍵字 如果結果失敗可以如何處理
Collections 清單使用
DateTime 日期與時間使用
Dialogs 停止執行 執行手動操作
String 轉換文字 長度確認 結合文字 頁面文字必須符合
